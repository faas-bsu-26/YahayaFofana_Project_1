
  # Work-Health Tracker App

  This is a code bundle for Work-Health Tracker App. The original project is available at https://www.figma.com/design/sgCa5rnODUKF368svALxln/Work-Health-Tracker-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

    ## Deploying to GitHub Pages

    This project is configured to deploy as a GitHub Pages project site for:
    `https://faas-bsu-26.github.io/Project_1_Fofana/`

    1. Push your latest changes to `main`.
    2. Run `npm run deploy`.
    3. In GitHub: **Settings → Pages → Build and deployment → Source**
      set source to **Deploy from a branch**, then select branch **gh-pages** and folder **/(root)**.

    After that, each time you run `npm run deploy`, the `gh-pages` branch updates and Pages republishes.
