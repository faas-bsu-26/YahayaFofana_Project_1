# Work-Health Tracker App

This is a code bundle for Work-Health Tracker App. The original project is available at https://www.figma.com/design/sgCa5rnODUKF368svALxln/Work-Health-Tracker-App.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deploying to GitHub Pages (Automatic)

This repository includes a workflow at `.github/workflows/deploy-pages.yml` that deploys automatically on every push to `main`.

1. Push your latest code to `main`.
2. In GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.

Your site will be published at:
`https://faas-bsu-26.github.io/Project_1_Fofana/`

## Optional manual deploy

Manual deploy is still available with:

- `npm run deploy`
